### Quick Deploy to Vercel

1. **Import to Vercel:**
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import this repository
   - Deploy automatically

2. **Or use Vercel CLI:**
   ```bash
   npm i -g vercel
   vercel
   ```
